### Étape 1 : Lire (15 min)
```bash
→ Ouvrir TESTS_AUTOMATION_README.md
→ Lire la section "Quick Start"
```

### Étape 2 : Consulter la roadmap (20 min)
```bash
→ Ouvrir docs/automation-checklist.md
→ Suivre Phase 1 (Setup)
```

### Étape 3 : Installer (30 min)
```bash
npm install -D @playwright/test
npx playwright install
```

### Étape 4 : Copier le code (15 min)
```bash
→ Copier e2e/comprehensive-test-suite.spec.ts
→ Adapter URLs et credentials
```

### Étape 5 : Lancer les tests (15 min)
```bash
npx playwright test
npx playwright show-report
```

### Étape 6 : Progresser (semaines 1-4)
```bash
→ Suivre Phase 2-8 de la checklist
→ Consulter docs/test-cases-fonctionnels.md pour plus de cas
→ Réutiliser snippets de e2e/SNIPPETS.md
```
